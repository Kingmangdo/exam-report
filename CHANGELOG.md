# 변경 이력 (Changelog)

이 파일은 프로젝트 진행 중 변경된 사항들을 추적합니다.

## [1.3] - 2026년 1월 (모든 상세 요구사항 반영 완료)

### 확인됨 (완료)
- ✅ 점수 계산: 시험마다 총 문제수 다름, 과제/수업태도 100점 만점, 총점 합계+평균, 가중치 25% 동일
- ✅ 성적표 인증: 학생이름 + 학부모 핸드폰 뒷4자리
- ✅ 링크 유효기간: 발송 후 7일
- ✅ 성적표 디자인: 심플한 파란색 무거운 톤, 로고 포함, 단어시험 84점 이하 빨간색 표기
- ✅ 성적표 정보: 반 평균 포함, 이전 성적 비교, 코멘트, 종합 문구 (원장 입력 가능)
- ✅ 시스템 운영: 담당자 4명, 학생 100-150명, 하루 50명 성적 입력
- ✅ 예외 처리: 연락처 없음 알림, 발송 실패 1회 재시도+알림, 중복 입력 마지막만 유지

### 개발 준비 완료
- 모든 요구사항 확인 완료
- 개발 시작 가능 상태

## [1.2] - 2026년 1월 (상세 요구사항 반영)

### 확인됨
- ✅ 카카오톡 발송 방식: 링크로 보내기
- ✅ 발송 시점: 성적 입력 즉시 자동 발송
- ✅ 알림톡 메시지 템플릿 확정 (날짜 형식: yy-mm-dd)
- ✅ 평가 영역 4개 확정: RT점수, 단어시험, 과제점수, 수업태도
- ✅ RT점수/단어시험: 총 문제수/맞춘 문제수로 계산
- ✅ 단어시험 84점 이하 시 "재시험" 표기

## [1.1] - 2026년 1월 (카카오톡 발송 기능 추가)

### 추가됨
- 카카오톡 알림톡 발송 기능 (핵심 기능)
- 학부모 연락처 관리 기능
- 발송 이력 관리 기능
- 비개발자 친화적 UI/UX 강화
- Excel/CSV 일괄 등록 기능
- 데이터 백업/복원 기능
- 설정 관리 페이지 (API 키 설정 등)

### 변경됨
- 프로젝트 목적: 성적표를 학부모에게 카카오톡으로 발송하는 것이 최종 목적
- 개발 기간: 15-20일 → 18-23일 (카카오톡 발송 기능 추가로 인한 연장)
- MVP 범위: 카카오톡 발송 기능 포함

### 확인 필요
- 카카오톡 알림톡 발송 업체 정보
- 성적표 발송 방식 (링크/이미지/PDF)
- 발송 시점 (자동/수동)
- 알림톡 템플릿 구성
