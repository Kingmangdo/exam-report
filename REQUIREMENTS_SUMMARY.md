# 독강영어학원 데일리 성적표 시스템 - 요구사항 요약

## 📌 핵심 기능

### 1. 성적 입력 및 관리
- **평가 영역**: RT점수, 단어시험, 과제점수, 수업태도 (4개)
- **점수 계산**:
  - RT점수/단어시험: 총 문제수/맞춘 문제수 (시험마다 총 문제수 다름)
  - 과제점수/수업태도: 0-100점 만점
  - 총점: 4개 영역 합계 + 평균 (가중치 25% 동일)
- **특이사항**: 단어시험 84점 이하 시 빨간색으로 "재시험" 표기
- **중복 입력**: 같은 날짜 같은 학생 중복 입력 시 마지막만 유지

### 2. 카카오톡 알림톡 발송
- **발송 방식**: 링크로 보내기 (웹 페이지 링크)
- **발송 시점**: 성적 입력 즉시 자동 발송
- **알림톡 메시지**:
  ```
  안녕하세요. 독강영어학원입니다.
  [학생이름]의 [날짜] 영어 성적표가 도착했습니다. 확인해주세요.
  ```
  - 날짜 형식: yy-mm-dd
- **예외 처리**:
  - 학부모 연락처 없음 → 별도 알림 표시
  - 발송 실패 → 1회 재시도 + 별도 알림 표시

### 3. 성적표 보기 (학부모용)
- **인증 방식**: 학생이름 + 학부모 핸드폰 뒷4자리 입력
- **유효기간**: 발송 후 7일
- **표시 정보**:
  - 학생 이름, 날짜
  - 영역별 점수 (RT점수, 단어시험, 과제점수, 수업태도)
  - 총점 (합계 + 평균)
  - 반 평균 점수
  - 이전 성적과 비교 (상승/하락)
  - 코멘트/평가 문구
  - 종합 문구 (원장이 직접 입력)

### 4. 성적표 디자인
- **스타일**: 심플하고 깔끔한 스타일
- **색상**: 파란색 중 무거운 톤
- **로고**: 학원 로고 포함
- **특이사항**: 단어시험 84점 이하 시 빨간색 표기

### 5. 추가 기능
- **종합 문구 관리**: 원장이 직접 입력/수정 가능
- **반 평균 계산**: 자동 계산 및 표시
- **이전 성적 비교**: 상승/하락 화살표 및 점수 차이 표시
- **알림 관리**: 발송 실패, 연락처 없음 등 별도 알림 표시

## 📊 시스템 규모

- **사용자 수**: 성적 입력 담당자 4명
- **학생 수**: 100-150명 (유동적)
- **하루 평균 성적 입력**: 50명

## 🔒 보안 및 접근 제어

- 성적표 링크는 학생이름 + 학부모 핸드폰 뒷4자리로 인증
- 링크 유효기간: 발송 후 7일
- 만료된 링크 접근 시 안내 메시지
