# 독강영어학원 데일리 성적표 시스템

학생들의 일일 영어 평가 성적을 관리하고, 학부모에게 카카오톡 알림톡으로 성적표를 자동 발송하는 웹 기반 시스템입니다.

## 🚀 기술 스택

- **프론트엔드**: Vue.js 3 + Vite + Tailwind CSS
- **백엔드**: Node.js + Express
- **데이터베이스**: SQLite
- **배포**: Vercel (프론트엔드), Railway (백엔드)

## 📁 프로젝트 구조

```
exam-report/
├── frontend/          # Vue.js 프론트엔드
├── backend/           # Node.js + Express 백엔드
├── database/          # SQLite 데이터베이스 파일
├── docs/              # 문서
└── README.md
```

## 🛠️ 개발 환경 설정

### 프론트엔드 실행
```bash
cd frontend
npm install
npm run dev
```

### 백엔드 실행
```bash
cd backend
npm install
npm run dev
```

## 📝 주요 기능

- 학생 정보 관리
- 일일 성적 입력 및 조회
- 성적표 생성 및 카카오톡 알림톡 발송
- 성적 통계 및 시각화
- 비개발자 친화적 UI/UX
