## 🚀 적용 방법

### 백엔드 서버 재시작

백엔드 터미널에서:
1. `Ctrl + C`로 서버 종료
2. 다시 실행:
   ```cmd
   npm.cmd run dev
   ```

프론트엔드는 자동으로 새로고침됩니다 (Vite 핫 리로드).

---

## ✅ 테스트

다시 성적을 입력해보세요:

1. 학생 선택
2. 날짜 입력 (yy-mm-dd 형식)
3. RT 점수 입력 (총 문제수, 맞춘 문제수)
4. 단어시험 점수 입력
5. 과제점수, 수업태도 입력
6. 저장 버튼 클릭

이제 정상적으로 저장될 것입니다!

---

## 💡 참고

- RT 점수나 단어시험 점수를 입력하지 않아도 저장됩니다 (0점으로 처리)
- 같은 날짜에 같은 학생의 성적을 다시 입력하면 업데이트됩니다
- 반 평균은 같은 반, 같은 날짜의 모든 성적을 포함하여 계산됩니다
