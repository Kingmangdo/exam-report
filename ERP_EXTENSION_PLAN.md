# 독강 ERP 확장 프로젝트 (Phase 2) 수행 계획서

## 1. 프로젝트 개요
기존 '독강영어학원 성적표 시스템'을 기반으로 학원 운영 전반을 관리할 수 있는 ERP(학원 운영 솔루션)로 확장

## 2. 주요 개발 목표
- **학생 정보 고도화**: 학교, 담당 강사 정보 추가 및 불필요 항목 제거
- **상담일지 모듈**: 학생별 상담 내역 기록 및 히스토리 관리
- **권한 관리**: 관리자(원장)와 강사의 메뉴 및 데이터 접근 권한 분리
- **수납 관리**: 학원비 결제 현황 및 미납자 관리 (관리자 전용)

## 3. 상세 개발 단계 및 기록

### 📅 Phase 2-1: 학생 정보 및 상담일지 기초 (완료)
- [x] **학생 정보 필드 수정** (학교, 담임 선생님 추가 / 생년월일, 주소 제거)
- [x] **상담일지 테이블 설계** (Supabase)
- [x] **상담일지 API 개발** (CRUD)
- [x] **학생 관리 UI 업데이트**

### 📅 Phase 2-2: 권한 및 수납 관리 (완료)
- [x] **사용자(관리자/강사) 테이블 및 로그인 시스템** (JWT 인증)
- [x] **학원비 결제 관리 모듈 개발** (CRUD)
- [x] **권한별 메뉴 필터링 적용** (관리자 전용 메뉴 숨김 처리)
